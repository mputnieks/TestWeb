<div>

The <div> tag defines a division or a section in an HTML document.

The <div> element is often used as a container for other HTML elements to style them with CSS or to perform certain tasks with JavaScript.

By default, browsers always place a line break before and after the <div> element. However, this can be changed with CSS. A way to do that is using display: inline; property.

<span>

The <span> tag is used to group inline-elements in a document.

The <span> tag provides no visual change by itself, however, it is a simple way to hook specific properties to just a part of text or document.
